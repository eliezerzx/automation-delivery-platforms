## ✨ Funcionalidades Principais

* **🖥️ Interface Moderna:** Design inspirado no Figma com modo escuro e botões estilizados.
* **📝 Editor de Dados Integrado:** Altere seus arquivos de `pizza.txt` e `complemento.txt` sem sair do programa.
* **🤖 Automação com PyAutoGUI:** Execução precisa de tarefas como criação de cardápios, complementos e aplicação de descontos.
* **📜 Console de Logs em Tempo Real:** Terminal estilo VS Code integrado para acompanhar cada passo da automação.
* **🚨 Botão de Pânico (Safety First):** Atalho global `Ctrl + 0` para encerrar o programa instantaneamente em caso de erro.
* **📊 Barra de Progresso:** Feedback visual do tempo de espera antes de cada disparo.

## 🛠️ Tecnologias Utilizadas

* [Python](https://www.python.org/) - Linguagem base.
* [CustomTkinter](https://github.com/TomSchimansky/CustomTkinter) - Interface moderna e responsiva.
* [PyAutoGUI](https://pyautogui.readthedocs.io/) - Automação de interface (Mouse e Teclado).
* [Keyboard](https://github.com/boppreh/keyboard) - Gerenciamento de atalhos globais.
